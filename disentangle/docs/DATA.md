# Data and licensing — disentanglement sub-project (paper 3)

This sub-project measures representation geometry; it trains no models and ships no
data. It reads ground-truth morphological labels from three sources, each obtained
and regenerated locally under its own licence. Nothing derived from them is
committed.

## SEDRA (Syriac)

Factor labels for Syriac (lexeme, number, gender, state) and the vocalised
(`western`) and consonantal (`syriac`) surface forms come from **SEDRA**, the
Syriac Electronic Data Research Archive (Beth Mardutho). The records are committed
under `corpora/sedra_cache/` (regenerable with `corpora.sedra_scrape`) and this
package reads them from there (via `corpora.SEDRA_WORD_DIR`) — it keeps no separate copy.

SEDRA is distributed for academic and personal use with restrictions: no
redistribution of altered versions, and the work must cite Kiraz (see
`corpora.SEDRA_CITATION`). We ship both the regeneration code and, for
reproducibility, the SEDRA IV cache itself; observe SEDRA's terms if you reuse it.

> Kiraz, G. A. (1994). *Automatic Concordance Generation of Syriac Texts.* VI
> Symposium Syriacum, Orientalia Christiana Analecta 247: 461–475.

## UniMorph (Arabic, Hebrew)

Cross-lingual replication uses **UniMorph** inflection tables for Arabic (`ara`)
and Hebrew (`heb`): `lemma <TAB> form <TAB> features`. `disentangle/unimorph.py`
downloads them on demand into `disentangle/unimorph_cache/` (git-ignored). UniMorph
is released under **CC BY-SA 3.0**.

Arabic UniMorph forms carry full diacritics, which we strip to obtain a consonantal
control; Hebrew forms are unvocalised, so for Hebrew the consonantal form equals the
surface form (the control is vacuous, and the code marks it so).

> McCarthy, A. D., et al. (2020). *UniMorph 3.0: Universal Morphology.* LREC.

## Encoders

All encoders are used **frozen**, off the shelf, and are downloaded from the
Hugging Face Hub on first use:

| Language | Model | Licence |
| --- | --- | --- |
| Syriac | `google/canine-c` (byte; also a random-init control) | Apache-2.0 |
| Syriac | same + local Syriac LoRA adapter (from `neural/`) | Apache-2.0 |
| Syriac | `onlplab/alephbert-base` on Syriac→Hebrew transliteration | — |
| Arabic | `CAMeL-Lab/bert-base-arabic-camelbert-ca` | Apache-2.0 |
| Hebrew | `onlplab/alephbert-base` | — |

Transport to the Hub is verified through the OS trust store via `truststore`; no
certificate verification is disabled.

## What is committed

Only code, the rendered LaTeX tables (`paper/tables/TD*.tex`), and the figures
(`paper/figures/*.pdf`). The results manifest (`paper/results.json`), the UniMorph
cache, and the SEDRA cache are git-ignored and regenerated by
`python -m disentangle.results --compute all`.
